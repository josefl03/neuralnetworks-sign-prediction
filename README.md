# Neural Networks Sign Prediction DEMO
 
This is the repository for the Sign Prediction DEMO

# Installation

Install Python 3 (Tested with Python 3.12.3) and PIP

Then run:
```shell
pip install -r requirements.txt
```

And execute the main.py file:
```shell
python3 main.py
```
